ВНИМАНИЕ!
В врестке используются SVG спрайты, поэтому просто открытие html файла в браузере не подойдет. Необходимо запускать через веб сервер (OpenServer например). Есть некоторые несоответсвия с исходным макетом. Например реализация карт на странице. Сделал их просто скриншотами, чтобы не тратить много времени на яндекс карты. Также некоторые места с контеном просто продублированы, так как с тех.стороны не было смысла делать их.
Все стили написаны на SCSS. В роли слайдера используется SwiperJS. Для реализации анимации с привязкой к скроллу используется библиотека GSAP + плагин ScrollTrigger к нему. В роли сборщика проекта использовался GULP. Макет в фигме Репозитории этой верстки
Также есть проект на ReactJS, с которым Вы можете ознакомиться.
